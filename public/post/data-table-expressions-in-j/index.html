data.table - Expressions in j

Last updated on Sep 13, 2019 1 min read

Hi, in this post I show you how we can use j in data.table to compute a linear model by group. The general data.table syntax is [i, j, by]. You can read it as: rows, columns, grouped by what. The syntax will get clearer, once we use it.

Data

library(data.table)
## Warning: Paket 'data.table' wurde unter R Version 3.6.1 erstellt
dt <- data.table(x = rep(1 : 1000, 500), x1 = rnorm, y = runif(2e8))
